# AI-Powered Personal Finance Copilot

## Overview
An AI-driven financial intelligence system that analyzes transaction data, detects anomalies, and generates actionable insights.

## Features
- ML-based transaction categorization
- Hybrid anomaly detection (z-score + trend analysis)
- Multi-factor financial risk scoring
- Interactive dashboard with real-time analytics
- Explainable AI insights

## Tech Stack
- Backend: FastAPI, Python, MySQL
- ML: scikit-learn
- Frontend: React, Recharts
